# RocketLauncher

一个用来弹射玩家到空中的小插件。

## Features

- 在特定的位置设置特定角度的弹射点
- 传送到指定的弹射点
- 设置弹射的检测间隔
  - *高间隔意味着高精度，但也有损失性能的代价*

## Commands

- /rocketlauncher add [pointname] [from|to]
- /rocketlauncher remove [pointname]
- /rocketlauncher list
- /rocketlauncher tp [pointname]
- /rocketlauncher toggle [period]
- /rocketlauncher toggle
